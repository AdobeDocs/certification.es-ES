# Recorrido de certificación para Adobe [!DNL Workfront] Desarrollador principal

Aprenda a convertirse en un profesional certificado de Adobe Developer.

## Experiencia mínima

Debe tener:

* Al menos 3 - 6 meses de experiencia práctica con Adobe [!DNL Workfront]
* Experiencia con Adobe [!DNL Workfront] funcionalidad y conectividad mediante admisión, aprobación, planificación, ejecución y reporting
* Comprensión a fondo del Adobe [!DNL Workfront] tipos de objeto, incluidos formularios personalizados y flujos de trabajo de aprobación, plantillas, duraciones, esfuerzo de trabajo/horas planificadas
* Comprensión a fondo del Adobe [!DNL Workfront] Configuración, incluidos los niveles de acceso, las plantillas de diseño, las preferencias del proyecto y otros métodos para lograr la simplicidad del diseño del usuario final
* Comprensión y experiencia con la asignación de procesos
* Comprensión de diferentes Adobes [!DNL Workfront] personalidades y cómo participar en la asistencia

Y debería tener:

* Familiaridad con [!DNL Workfront] API y explorador de API, pero no necesariamente competente en llamadas de API
* La capacidad de usar llamadas de API personalizadas para trabajar con [!DNL Workfront] y otras API
* Familiaridad con el Adobe [!DNL Workfront]
* Familiaridad con conectores nativos y otro Adobe [!DNL Workfront] productos y servicios
* Familiaridad con la administración de proyectos, incluidos los conceptos de agilidad y cascada y de gestión de recursos

>[!NOTE]
>
>Su certificación es válida durante dos años a partir del momento de su aprobación. Se recomienda utilizar la versión más reciente cuando sea posible, ya que cubre las capacidades de los productos más recientes.

**Audiencia prevista:**

* Consultor técnico
* Administrador del sistema
* Administrador de proyectos
* Coordinador de proyectos
* Estratégico
* Administrador de recursos
* Analista
* Especialista en TI
* [!DNL Workfront] asesor de implementación/cliente potencial

**Detalles del examen:**

* Nivel: Profesional (0-12 meses de experiencia)
* Puntuación de paso: 44/70
* Tiempo: 140 minutos
* Envío: Procesado en línea (requiere acceso a la cámara)
* Idiomas disponibles: Inglés
* Costo: $125 (global) / $95 (India)
* ID del examen: AD0-E905

**Autoevaluación de la preparación**

Complete el [cuestionario de preparación en línea](https://scorpion.caveon.com/launchpad/ad-q-e905-readiness-questionnaire-for-adobe-workfront-core-developer-professional/ad-q-e905-readiness-questionnaire-for-adobe-workfront-core-developer-professional){target="_blank"} para ver si su experiencia actual coincide con la cualificación mínima recomendada de los candidatos.

**Objetivos y ámbito del examen**

Sección 1. Estrategia y admisión (24%)

* Usar portafolios y programas
* Usar colas de solicitud
* Usar formularios personalizados
* Usar aprobaciones básicas

Sección 2. Presupuesto y planificación (11%)

* Usar el planificador de recursos
* Dados un escenario/conjunto de requisitos, aplique las configuraciones para configurar una plantilla de proyecto

Sección 3. Experiencia del usuario (25%)

* Ilustrar prácticas recomendadas para la administración de usuarios al crear la experiencia de usuario correcta
* En un escenario, aplique la configuración correcta para los derechos de uso compartido y acceso en todo el sistema
* Describa qué es un inicio rápido y reconozca cuándo utilizarlo
* Demostrar cómo crear y usar hojas de horas
* Describir el propósito y el valor de los conectores, complementos y complementos

Sección 4. Métricas e informes (16%)

* Describir conceptos básicos para crear, enviar y personalizar informes
* Explicar cómo crear y personalizar tableros
* Demostrar cómo configurar vistas funcionales de informes

Sección 5. Ejecución (24%)

* Demostrar cómo configurar flujos de trabajo de aprobación y prueba
* Reconocer el propósito de crear un problema para un proyecto
* Demostrar cómo personalizar estados de proyectos de nivel de grupo
* Identifique los atributos necesarios para habilitar Agile en [!DNL Workfront]
* Identificar las formas de entrar en tiempo [!DNL Workfront]
* Identifique los elementos del seguimiento financiero en [!DNL Workfront]

**Haga el examen de certificación**

Adobe utiliza dos proveedores para administrar los exámenes procesados, PSI y Ejemplos.

* **PSI** ofrece opciones de centro de pruebas y exámenes japoneses (si está disponible)
* **[!DNL Examity]** ofrece una opción en línea remota

Para programar, volver a programar o cancelar el examen, simplemente haga clic en el proveedor que desee y siga las instrucciones que aparecen en la pantalla. ¡Es así de simple!
