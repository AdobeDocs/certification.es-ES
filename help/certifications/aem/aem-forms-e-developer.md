# Recorrido de certificación para Adobe [!DNL Experience Manager Forms] Experto en desarrollo

Aprenda a convertirse en un certificado [!DNL Experience Manager Forms] Experto en desarrollo.

## Experiencia mínima

Debe tener:

* 1 a 3 años de experiencia con AEM Forms
* Al menos 5 años de experiencia laboral en la industria de TI
* La capacidad para liderar el desarrollo, la configuración, las pruebas y la implementación de soluciones de formularios personalizados
* La capacidad de configurar un formulario, traducir los requisitos comerciales en flujos de trabajo e integrarlos con sistemas externos
* Comprensión de los componentes básicos de AEM Forms

Además, debe estar familiarizado con las siguientes tecnologías y entornos:

* Diseñar plantillas XDP, plantillas XFA
* Creación de PDF forms (estático/dinámico, interactivo y no interactivo)
* Enlace de datos XML o de esquema
* Crear fragmento de formulario
* Crear un formulario adaptable
* Depuración de formularios complejos
* Configuración y realización del envío de formularios
* Creación e implementación de instancias de AEM para formularios adaptables
* Uso del modelo de formulario
* Capacidad de integración con servicios de datos
* Implementación del flujo de trabajo
* Comprensión básica de AEM área de trabajo de formularios (JEE) para crear procesos y servicios en directo breves y largos.
* Desarrollar clientlibs de javascript mediante la API de GuideBridge
* Basado en el signo v gobierno CAC sabe qué servidor o escritorio de entorno de AEM Forms está basado para usar
* Comprensión básica de Java/J2EE y JavaScript para escribir la lógica y llamar a diferentes servicios.
* Rellenar previamente un formulario adaptable
* Comprensión o conocimiento básicos de OSGi, componentes de Sling
* Uso de servicios de documentos
* AEM Designer, AEM Workbench, AEM workspace
* Consola de formularios adaptables
* Herramientas IDE de Java
* Maven
* Herramientas de GIT/versiones

>[!NOTE]
>
>Su certificación es válida durante dos años a partir del momento de su aprobación. Se recomienda utilizar la versión más reciente cuando sea posible, ya que cubre las capacidades de los productos más recientes.

* Puntuación de paso: 27/50
* Tiempo: 100 minutos
* Envío: Procesado en línea (requiere acceso a la cámara) o procesado en el centro de pruebas
* Idiomas disponibles: Inglés
* Costo: $225 (global) / $150 (India)
* ID del examen: AD0-E125

**Autoevaluación de la preparación**

Complete el [cuestionario de preparación en línea](https://scorpion.caveon.com/launchpad/ad-q-e129-readiness-questionnaire-for-adobe-aem-assets-developer-professional-exam-copy-x38d6m/ad-q-e125-readiness-questionnaire-for-adobe-aem-forms-developer-expert-exam){target="_blank"} para ver si su experiencia actual coincide con la cualificación mínima recomendada de los candidatos.

**Objetivos y ámbito del examen**

Sección 1: Creación y administración de Forms (36%)

* Generar Forms
* Configuración del modelo de datos de Forms
* Personalizar AEM Forms
* Desarrollar y utilizar Javascript ClientLib aprovechando la API de GuideBridge

Sección 2: Procesos y flujos de trabajo (34%)

* Diseño y organización de un flujo de trabajo de aprobación de formularios
* Desarrollo de un paquete OSGi mediante la API de servicios de documentos
* Programar diferentes procesos por lotes asincrónicos usando el extremo Watchfolder
* Producir resultados diferentes del diseño XFA/XDP
* Manipulación del PDF mediante Forms o los servicios del ensamblador
* Implementación de servicios JDBC

Sección 3: Administración y solución de problemas (18%)

* Solucionar problemas y diferenciar archivos de registro
* Realizar actividades de administración
* Aplicar Service Packs
* Realizar AEM configuración de Dispatcher para Forms
* Desplazamiento por diferentes consolas

Sección 4: Puntos finales e integraciones de API (8%)

* Integración con Adobe Experience Cloud
* Generación y consumo de varios servicios de extremo

Sección 5: Comunicación interactiva (4%)

* Diseñar, demostrar e implementar procesos para la comunicación interactiva

**Realizar una prueba de práctica**

Se recomienda realizar la prueba de práctica gratuita antes de comenzar. Tendrá acceso sin restricciones a todas las pruebas de práctica disponibles para los exámenes de nivel de experto de Adobe.

Nuestras pruebas de práctica se desarrollan desde el mismo modelo que los exámenes en directo. La prueba de práctica le ayudará a comprender los temas tratados en el examen en vivo.

Recuerde que aprobar una prueba de práctica no garantiza que se aprobará el examen de certificación.

Vaya a la [Portal de preparación de examen de certificación de Adobe](https://www.certmetrics.com/adobe/candidate/gmetrix_sso.aspx){target="_blank"} ahora, para realizar una prueba de práctica.

**Haga el examen de certificación**

Adobe utiliza dos proveedores para administrar los exámenes procesados, PSI y Ejemplos.

* **PSI** ofrece opciones de centro de pruebas y exámenes japoneses (si está disponible)
* **[!DNL Examity]** ofrece una opción en línea remota

Para programar, volver a programar o cancelar el examen, simplemente haga clic en el proveedor que desee y siga las instrucciones que aparecen en la pantalla. ¡Es así de simple!
